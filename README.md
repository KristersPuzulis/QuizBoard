# Quiz Board Builder

A browser-based Jeopardy-style quiz board app built with HTML, CSS, and JavaScript.

## Features
- Play mode with team score tracking
- Editable quiz board with categories, questions, answers, and Daily Doubles
- Adjustable board size up to 10 × 10
- Add and remove teams, up to 10 teams
- Save boards in the browser
- Export and import boards as TXT files
- Add image, audio, or video media to questions
- Ready-made board templates and saved board manager
- Responsive layout for different screen sizes

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- LocalStorage
- IndexedDB

## Why I Built This
Me and my friends wanted to play Jeopardy-style quiz game, but all the options online require a payment for extra features, so I decided to make one myself

## What I Learned
- Browser storage using LocalStorage and IndexedDB
- File import/export with JavaScript
- Responsive UI layout
- Managing app state across multiple screens
- Building reusable board data structures
